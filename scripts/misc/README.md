# Misc Scripts

通用/环境维护脚本。

## 主要脚本

- `lab_healthcheck.py` — 实验室环境健康检查
- `first_run_check.py` — 首次打开仓库时检查工作区与 `reverse_lab_tools` MCP 配置
- `start_here.ps1` — Windows 双击入口 `START_HERE.cmd` 调用的首次设置助手
- `ai_toolcheck.py` — AI 工具可用性检查
- `ai_tool.py` — AI 工具路由器
- `ai_context.py` — 任务上下文生成
- `ai_finding.py` — 发现记录管理
